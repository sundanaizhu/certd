# 彩虹易支付配置

彩虹易支付是一款非常流行的php聚合支付系统。

## 配置步骤

1. 获取商户ID、商户密钥

   * 登录彩虹易支付平台
   * 进入用户中心：https://xxxxxx.com/user/userinfo.php?mod=api
   * 点击API信息
   * 可以复制：接口地址、商户ID、商户密钥（key）
   * 点击查看文档，了解支持的签名类型，一般为MD5

2. 进入Certd后台配置彩虹易支付

   * 进入“系统”->"设置"->“支付设置”
   * 启用彩虹易支付，选择“彩虹易支付配置”，点击添加
   * 填写接口地址、商户ID、商户密钥、签名方式等信息即可。
